Python Scraper (with Selenium)
================================

> [Aeon Digital](http://www.aeondigital.com.br)

&nbsp;

Python scripts for scraping product data, handling pagination, extracting product details, and downloading images.  
All scripts are designed to work with **Selenium + ChromeDriver** and can be customized with CSS selectors for different websites.

## Usage Examples

### 1. Scraping product links with pagination (`001_pagination.py`)

This script navigates through paginated product listings, extracts product names and links, and saves them to a CSV file.

```bash
python 001_pagination.py "https://example.com" "/products" "div.product-card" "h2.product-title" "a.product-link" "a.next-page" "products.csv"
```

Output:
- `products.csv` containing:
  ```
  "Name","Link"
  "Product A","https://example.com/product-a"
  "Product B","https://example.com/product-b"
  ...
  ```

&nbsp;
&nbsp;


### 2. Scraping product details (`002_product.py`)

This script reads the CSV generated by `001_pagination.py`, visits each product page, and extracts details based on CSS selectors.

```bash
python 002_product.py products.csv details.csv Price "span.price" innerText Image "div.product-image img" src Gallery "div.gallery img" src[]
```

Output:
- `details.csv` containing product details with fields like price, image, and gallery.

&nbsp;
&nbsp;


### 3. Downloading product images (`003_getimages.py`)

This script downloads images listed in a CSV column (single URL or JSON array of URLs) and saves them locally with sequential numbering.

```bash
python 003_getimages.py details.csv Gallery arrayJSON images/ Name
```

Output:
- Images saved in the `images/` folder with filenames like:
  ```
  00001-ProductA-img1.jpg
  00002-ProductA-img2.jpg
  00003-ProductB-img1.jpg
  ```

## Notes

- Adjust CSS selectors according to the target website structure.  
- Use `arrayJSON` mode when a CSV cell contains multiple image URLs in JSON format.  
- Always run the Selenium test (`src/test/selenium.py`) before scraping to confirm environment setup.
